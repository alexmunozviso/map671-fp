Data Sources

I built my Bigfoot sightings database using the 2016 updated creature sighting database from The Bigfoot Field Research Organization (BFRO) summarized by Mangani's Bigfoot Maps. 2022 United States county and state boundaries were obtained from the United States Census Bureau website.

Other Sources

The Bigfoot miniature used as the webmap report symbol and at the header of this site is a slightly modified version of a vector downladed from Freepik.

Tools used in this project

  • Visual Studio Code was used to create the Bigfoot Sighting Database.
  • QGIS was used to geoprocess, analyze, and represent the geodata for this project.
  • Data is shown on Mapbox web map. If the map got popular it would require a fee to use.
  • Adobe Illustrator was used to edit the Bigfoot miniatures used for both the webmap and this site header.

Static Maps from the Project

Aside from the webmap available at the top of this page, I built three static maps for this project:
